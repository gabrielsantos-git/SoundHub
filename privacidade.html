Política de Privacidade

Versão 1.0 Última atualização: junho de 2025  ·  Em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018)
Esta Política de Privacidade explica como o SoundHub coleta, utiliza, armazena e protege seus dados pessoais. Leia com atenção antes de criar sua conta.

1. Quem é responsável pelos seus dados

O controlador dos dados é a organização que administra esta instalação do SoundHub. Em caso de dúvidas sobre seus dados, entre em contato pelo canal indicado na seção 10 desta política.

O SoundHub utiliza a infraestrutura de banco de dados e armazenamento de arquivos da Supabase Inc. (San Francisco, EUA), que atua como operadora de dados em conformidade com as diretrizes de privacidade aplicáveis.

2. Quais dados coletamos

Dado Finalidade Fornecido por
Nome completo Identificação do usuário no sistema Você, no cadastro
Endereço de e-mail Login e comunicação sobre sua conta Você, no cadastro
Senha Autenticação (armazenada criptografada com bcrypt) Você, no cadastro
Cargo / função Controle de acesso e permissões Definido pelo administrador
Data e hora do cadastro Registro de auditoria Gerado automaticamente
Data e hora do consentimento Comprovação de aceite desta política (LGPD) Gerado automaticamente
Endereço IP no cadastro Registro de segurança e conformidade Coletado automaticamente
Arquivos de mídia enviados Uso litúrgico e organizacional Você, via QR Code
Nome do remetente do arquivo Identificação do envio Você, no momento do envio
Não coletamos: dados de localização, dados de saúde, dados financeiros, dados biométricos, dados de menores sem consentimento dos responsáveis, nem qualquer dado sensível além dos listados acima.

3. Para que usamos seus dados

  • Criar e gerenciar sua conta de acesso ao sistema
  • Controlar permissões de acordo com sua função (sonoplasta, diretor, administrador)
  • Permitir o envio, aprovação e exibição de mídias para uso nos cultos e eventos
  • Garantir a segurança e rastreabilidade das ações no sistema
  • Cumprir obrigações legais e regulatórias, incluindo a LGPD

Seus dados não são usados para fins comerciais, publicidade ou compartilhados com terceiros para qualquer finalidade além das listadas acima.

4. Base legal para o tratamento

TratamentoBase legal (LGPD, Art. 7º)

Controle de permissões e cargos Legítimo interesse do controlador (inciso IX)
Armazenamento de IP e data de consentimento Cumprimento de obrigação legal (inciso II)

5. Compartilhamento de dados

Seus dados são armazenados nos servidores da Supabase Inc., que é a plataforma de banco de dados utilizada pelo SoundHub. A Supabase atua exclusivamente como operadora, processando os dados conforme as instruções do controlador.

Não vendemos, alugamos nem compartilhamos seus dados com nenhuma outra empresa, anunciante ou pessoa física, exceto quando exigido por lei ou ordem judicial.

7. Seus direitos como titular de dados

Nos termos da LGPD (Art. 18), você tem os seguintes direitos:

Acesso

Saber quais dados temos sobre você e obter uma cópia.

Correção

Corrigir dados incompletos, inexatos ou desatualizados.

Exclusão

Solicitar a exclusão de seus dados pessoais do sistema.

Portabilidade

Receber seus dados em formato legível para uso em outro serviço.

Revogação do consentimento

Retirar seu consentimento a qualquer momento, sem prejuízo dos tratamentos já realizados.

Informação

Saber com quais entidades seus dados foram compartilhados.

Para exercer qualquer um desses direitos, entre em contato conforme indicado na seção 10. Responderemos em até 15 dias úteis.

8. Como protegemos seus dados

  • Senhas armazenadas com hash criptográfico (bcrypt, fator 10)
  • Comunicação com o servidor via HTTPS (criptografia em trânsito)
  • Acesso ao banco de dados restrito por chaves de API protegidas
  • Controle de acesso baseado em funções (RBAC): cada usuário acessa apenas o que sua função permite
  • Tokens de sessão com expiração automática

Em caso de incidente de segurança que envolva seus dados, você será notificado conforme exigido pelo Art. 48 da LGPD.

9. Cookies e armazenamento local

O SoundHub utiliza o localStorage do navegador para armazenar o token de autenticação durante a sessão. Esse token é necessário para manter você logado enquanto usa o sistema.

Não utilizamos cookies de rastreamento, publicidade ou análise de comportamento.

10. Contato e encarregado de dados (DPO)

Fale com o responsável pelos seus dados

Para exercer seus direitos, tirar dúvidas ou reportar alguma situação relacionada à sua privacidade, entre em contato com o administrador do sistema:

Canal de contato: diretamente com o administrador da sua organização responsável pelo SoundHub.

Para questões relacionadas à ANPD (Autoridade Nacional de Proteção de Dados), acesse: www.gov.br/anpd

11. Atualizações desta política

Esta política pode ser atualizada periodicamente. Em caso de alterações relevantes, os usuários serão informados pelo sistema. A data da última atualização está sempre indicada no topo deste documento.

O histórico de versões aceitas por cada usuário é registrado no sistema para fins de conformidade.
